CHAR и VARCHAR служат символьными типами данных:

- **VARCHAR** используется для строк символов переменной длины
- **CHAR** используется для строк фиксированной длины

Например, `char(10)` может хранить только 10 символов и не сможет хранить строку любой другой длины (если попытаться добавить строку меньшей длины оставшееся место будет заполнено пробелами), тогда как `varchar(10)` может хранить строку любой длины до 10, т.е. например 6, 8 или 2.
