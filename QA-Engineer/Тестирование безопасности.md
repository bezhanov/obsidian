Инструменты для тестирования безопасности

Собрали приложения, которые используются для нахождения уязвимостей в компьютерных системах. Эти инструменты имитируют атаки на систему, чтобы определить потенциально слабые места и дают информацию о том, как уменьшить эти риски:

▫️Acunetix (https://www.acunetix.com/) — инструмент для постоянного автоматизированного сканирования
▫️ (https://www.acunetix.com/)Intruder (https://www.intruder.io/) — инструмент для автоматического сканирования
▫️ (https://www.acunetix.com/)Astra (https://crozdesk.com/software/astra/) — инструмент, позволяющий взглянуть на безопасность с точки зрения хакера. Создан по стандартам OWASP и SANS
▫️ (https://www.acunetix.com/)Metasploit (https://www.metasploit.com/) — определяет слабые места системы
▫️ (https://www.acunetix.com/)W3af (https://w3af.org/) — бесплатный open-source сканер
