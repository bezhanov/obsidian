### Итератор

**Итератор** — это объект, который поддерживает протокол итерации, т.е. реализует методы `__iter__()` и `__next__()`.

- **Метод `__iter__()`** возвращает сам итератор. Он необходим, чтобы объект мог использоваться в циклах, таких как `for`.
- **Метод `__next__()`** возвращает следующий элемент из последовательности. Когда элементы заканчиваются, метод должен вызвать исключение `StopIteration`.

Пример простого итератора:

```
class MyIterator:     
	def __init__(self, data):         
		self.data = data         
		self.index = 0      
	
	def __iter__(self):         
		return self      
	
	def __next__(self):         
		if self.index < len(self.data):             
			result = self.data[self.index]             
			self.index += 1             
			return result         
		else:             
			raise StopIteration  

# Использование итератора 
iterator = MyIterator([1, 2, 3]) 
for item in iterator:     
	print(item)
```

### Генератор

**Генератор** — это специальный вид итератора, который удобен для создания итераторов. Генераторы могут быть созданы с помощью выражений или функций с ключевым словом `yield`.

- **Функции-генераторы** — это функции, которые возвращают значения с помощью `yield`, а не `return`. В момент вызова такой функции выполнение функции приостанавливается на месте `yield`, сохраняя состояние, и возобновляется при следующем вызове метода `__next__()`.

Пример генератора:

```
def my_generator():     
	yield 1     
	yield 2     
	yield 3  
	
# Использование генератора 
for item in my_generator():     
	print(item)
```

**Генераторы** удобны, когда нужно итерировать по большому объему данных или по потенциально бесконечной последовательности, так как они вычисляют элементы по мере необходимости и не загружают все данные в память сразу.

#### Выражение-генератор

Кроме того, существует **выражение-генератор**, которое создает генератор «на лету». Пример:

```
gen_exp = (x * x for x in range(5)) 
for item in gen_exp:     
	print(item)
```

**Итераторы** и **генераторы** являются основными инструментами Python для работы с итеративными задачами, и их понимание важно для написания эффективного и производительного кода.
