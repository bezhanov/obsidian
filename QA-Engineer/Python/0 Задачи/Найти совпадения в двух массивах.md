Даны два массива с целыми числами nums1 и nums2 размера n и m соответственно.
Посчитайте:
- answer1: количества индексов і в массиве nums1 такое, что nums1[i] существует в nums2
- answer2 : количества индексов і в массиве nums2 такое, что nums2[i] существует в nums1
Верните [answer1, answer2].

```python
def count_equals(nums1, nums2):
	
	# Задаем множества, убираем повторяющие значения
	hash1 = set(nums1)
	hash2 = set(nums2)
	
	answer1 = 0
	answer2 = 0
	
	for num in nums1:
		if num in hash2:
			answer1 += 1
	for num in nums2:
		if num in hash1:
			answer2 += 1
			
	return [answer1, answer2]
```
