Дан целочисленный массив nums и число k.
Частота элемента х это количество раз, которое он содержится в массиве.
Массив называется хорошим, если частота каждого эелемента в нем меньше или равна k.
Верните длину самого длинного хорошего подмассива nums.

Input: nums = [1,2,3,1,2,3,1,2], k = 2
Output: 6
Input: nums = [1,2,1,2,1,2,1,2], k = 1
Output: 2
Input: nums = [5,5,5,5,5,5,5], k = 4
Output: 4


```python

def longest_good(nums, k):

	max_len = 0
	dict_count = defayltdict(int)
	left = 0 # левый индекс
	right = 0 # правый индекс
	
	for num in nums:
		dict_count[num] += 1 # добавляем число повторения для значения
		right += 1 # увеличиваем отрезок вправо
		if dict_count[num] > k: # повторений этого значения больше k в словаре
			max_len = max(max_len, (right - 1) - left) # фиксируем длиный отрезок
			while dict_count(num) > k: # нужно теперь уменьшать слева пока
				dict_count[nums[left]] -= 1
				left += 1
		max_len = max(max_len, (right - 1) - left)
```
