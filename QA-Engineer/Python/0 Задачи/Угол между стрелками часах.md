При наличии двух чисел `hour` (часы) и `minutes` (минуты), вернуть наименьший угол (в градусах), образованный между часовой `hour` и минутной `minute` стрелками.

Ответы в пределах `10-5` фактического значения будут приняты как правильные.

Ввод: `hour = 12, minutes = 30`

Вывод: `165`

**Пример 3:**

![369](https://7efaf40e-9773-4d49-9fd7-20d5d05d2563.selstorage.ru/img/TMCzSbdTrwWCfYX6.png)

Ввод: `hour = 3, minutes = 15`

Вывод: `7.5`

**Ограничения:**

`1 <= hour <= 12`

`0 <= minutes <= 59`


Решение:

Как движется минутная стрелка
- За 60 минут она проходит 360°.
- Значит за 1 минуту проходит: 360/60=6
- minute_angle = minutes × 6

Как движется часовая стрелка

- За 12 часов она проходит 360°.
- За 1 час: 360/12=30
- Но! Часовая стрелка двигается **плавно**, а не скачком раз в час.
- За 1 минуту она проходит: 30/60=0.5
- hour_angle = (hour % 12) × 30 + minutes × 0.5
- `hour % 12` нужен потому, что 12 часов = 0

```python 
def angleClock(self, hour: int, minutes: int) -> float:  
	# Угол минутной стрелки  
	minute_angle = minutes * 6  
  
	# Угол часовой стрелки  
	hour_angle = (hour % 12) * 30 + minutes * 0.5  
  
	# Разница между углами  
	angle = abs(hour_angle - minute_angle)  
  
	# Берем наименьший угол  
	return min(angle, 360 - angle)
```
