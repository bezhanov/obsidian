Дана строка s, состоящая из символов (0, 1), и число k. Верните True, если s содержит все двоичные коды длины k. Иначе верните False

Input: s = '00110110', k = 2
Output: False

```python
def all_binary_codes(s, k):

	hash_s = set()
	
	for i in range(len(s) - k + 1):
		hash_s.add(s[i: i + k])
	
	if len(hash_s) == 2 ** k
		return True
	
	return False
```
