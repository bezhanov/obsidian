PEP (Python Enchancement Proposal) - это предложения по развитию Python. Процесс PEP является основным механизмом для предложения новых возможностей и для документирования проектных решений, которые вошли в Python

PEP8 - это style guide того, как должен быть оформлен код, написанный на Python и к которому должны следовать все разработчики, которые пишут на этом языке
