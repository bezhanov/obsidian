Хэш-функция в Python — это функция, которая принимает на вход объект и возвращает уникальное числовое значение фиксированного размера, называемое хэш-кодом или хэшом. В Python для этого используется встроенная функция `hash()`.

Хэш-функции играют важную роль при работе со структурами данных, такими как множества (`set`) и словари (`dict`), где для эффективного поиска, вставки и удаления элементов используются хэш-коды.

### Основные характеристики хэш-функции:

1. **Детерминированность**: Для одного и того же объекта хэш-функция всегда возвращает одинаковое значение в течение одного сеанса работы программы.
2. **Эффективность**: Хэш-функция должна быстро вычислять хэш-код.
3. **Однонаправленность**: По хэш-коду сложно восстановить исходный объект.
4. **Коллизии**: Разные объекты могут иметь одинаковый хэш-код, но хэш-функция должна минимизировать вероятность таких коллизий.

### Пример использования:

```
`# Пример хэширования строки
string_hash = hash("Hello, World!")
print(string_hash)  

# Пример хэширования числа 
number_hash = hash(123) 
print(number_hash)`
```

### Особенности:

- Встроенная функция `hash()` работает только с неизменяемыми объектами (такими как числа, строки, кортежи), потому что хэш-код должен оставаться постоянным для данного объекта.
- Множества и словари используют хэш-коды для эффективной организации данных, что позволяет им обеспечивать операции поиска за время, близкое к постоянному.

### Важно:

Начиная с версии Python 3.3, хэш-коды для строк и некоторых других объектов случайным образом изменяются при каждом запуске интерпретатора Python. Это сделано для защиты от атак, использующих предсказуемые хэш-значения. Однако для неизменяемых типов данных, таких как целые числа, хэш-коды остаются одинаковыми в разных запусках.
